# Caféine

Caféine is a simple coffee shop that offers quality coffee.

🔗 **Live Demo:** https://markjasperrojas.github.io/restaurant-page/

This project was inspired by the Restaurant Page Project from [The Odin Project](https://www.theodinproject.com/) and it was built from scratch. Through this project, I learned and practiced ES6 modules and Webpack.
